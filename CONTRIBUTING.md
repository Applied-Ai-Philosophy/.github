# Contributing to Applied AI Philosophy

Thank you for your interest in contributing to Applied AI Philosophy research. This document outlines how researchers, developers, and curious minds can participate.

## Ways to Contribute

### üî¨ Research Contributions

**Empirical work**
- Replicate SRT Protocol experiments with different AI systems
- Contribute datasets to savant-fnc or fnc-lab
- Report anomalous AI behaviours (potential Turn-5 Events)

**Theoretical work**
- Extend FNC framework to new domains
- Propose refinements to existing papers
- Cross-reference with your own research

**To contribute research:**
1. Open a [Discussion](https://github.com/orgs/Applied-Ai-Philosophy/discussions) describing your work
2. We'll identify the appropriate repository
3. Submit via Pull Request or collaborative authorship

### üíª Code Contributions

**What we need:**
- Analysis tools for FNC scoring
- Visualisation improvements
- Documentation and examples
- Bug fixes and performance improvements

**Process:**
1. Check existing [Issues](https://github.com/Applied-Ai-Philosophy/fnc-lab/issues) or open a new one
2. Fork the relevant repository
3. Create a feature branch (`git checkout -b feature/your-feature`)
4. Commit with clear messages
5. Open a Pull Request

### üìù Documentation

- Fix typos or unclear explanations
- Translate documentation
- Add examples or tutorials
- Improve README files

### üí¨ Community

- Answer questions in Discussions
- Share your experience using FNC
- Connect us with relevant researchers
- Cite our work in your publications

## Guidelines

### Academic Integrity

- Cite sources appropriately
- Acknowledge uncertainties
- Don't overstate claims
- Respect intellectual property

### Code Standards

- Write clear, commented code
- Include tests where appropriate
- Follow existing code style
- Document your changes

### Communication

- Be respectful and constructive
- Stay on topic
- Assume good faith
- Welcome newcomers
